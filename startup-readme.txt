=============================
NOTAS SOBRE O USO DO SISTEMA 
=============================

Para executar o programa, basta executar o arquivo startup.bat

OBS: O sistema está preparado para executar na pasta 
"c:\openreports-tomcat", mas essa configuração pode ser alterada pelo 
usuário "admin" senha "admin" em http://localhost:8080/openreports

Na pasta "projeto-eclipse" há o projeto "openreports", pronto para ser 
importado no Eclipse, facilitando a customização dos fontes. 

Se houver problemas com a funcionalidade de envio de e-mails agendados, 
deve-se editar o arquivo MailProvider.java no Eclipse e recompilar o 
sistema.


=============================
NOTES FOR USING THIS SOFTWARE
=============================

To run this software, execute file startup.bat

OBS: The system is ready to be executed on directory 
"c:\openreports-tomcat", but this can be reconfigured by logging with 
user "admin" password "admin", using URL http://localhost:8080/openreports

In directory "projeto-eclipse" there is a project named "openreports", 
ready to be import from Eclipse, making the source easy to be customized.

If you have problems with e-mail sending functionality, 
try to edit directly file MailProvider.java using Eclipse amd recompile 
the system. Some providers such as gmail require special parameters.
